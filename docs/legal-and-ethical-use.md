# Legal and Ethical Use

WebShield Security Scanner is a defensive security tool.

## Allowed Use

Use WebShield when you have permission to test the target system, such as:

- Your own website
- Your company's internal systems
- Client systems with written authorization
- Local development environments
- Capture-the-flag or intentionally vulnerable training labs

## Prohibited Use

Do not use WebShield to test systems where you do not have authorization.

Do not use this project for:

- Unauthorized vulnerability scanning
- Credential attacks
- Brute forcing
- Exploitation
- WAF bypass
- Persistence or post-exploitation activity
- Public internet mass scanning
- Disruptive or destructive testing

## Product Policy

The public product should emphasize:

- Defensive audit
- Developer remediation
- Report generation
- CI/CD integration
- Safe configuration checks

Avoid marketing language such as:

- Hack any website
- Bypass security
- Exploit targets
- Steal credentials
- Attack automation

## Recommended Marketplace Description

WebShield Security Scanner helps developers audit their own web applications for common security misconfigurations, generate clear reports, and apply practical remediation guidance before production release.
