# Sponsorship

WebShield Security Scanner is a developer-first defensive security tool for auditing websites, APIs, and modern web applications.

Sponsorship helps keep the project maintained, documented, and practical for real developer workflows.

## What Sponsorship Supports

Funding helps support:

- New safe security checks
- Better report templates
- HTML and PDF export
- CI/CD integration examples
- ASP.NET Core remediation guides
- Angular hosting and frontend security guides
- Nginx, Docker, and Kubernetes hardening examples
- Documentation, sample projects, and release packaging
- Issue triage and project maintenance

## Suggested Sponsor Tiers

These are suggested tiers you can configure in GitHub Sponsors.

| Tier | Suggested price | Audience | Value |
|---|---:|---|---|
| Supporter | $3/month | Individual developers | Help maintain the project |
| Developer | $10/month | Developers using WebShield in side projects | Vote on next checks and roadmap items |
| Pro Supporter | $29/month | Freelancers and consultants | Early access to report templates and examples |
| Team Sponsor | $99/month | Small teams and agencies | Logo/name recognition and priority feature review |
| One-time Sponsor | $10+ one-time | Anyone | Support a release, feature, or documentation update |

## Recommended GitHub Sponsors Profile Copy

Use this short description on your GitHub Sponsors profile:

```text
I build developer-first tools for secure web application delivery. Sponsorship helps me maintain WebShield Security Scanner, add safe defensive checks, improve report generation, and create practical remediation guides for ASP.NET Core, Angular, APIs, Docker, Nginx, and Kubernetes.
```

## Sponsor Benefits Ideas

You can offer:

- Sponsor name in the README
- Early access to report templates
- Roadmap voting
- Priority issue review
- Sponsor-only examples
- Monthly development notes
- Recognition in release notes

Avoid promising security consulting or emergency support unless you are ready to deliver it consistently.

## Responsible Funding Statement

WebShield is funded to improve defensive security workflows. The project does not aim to provide exploitation, brute force, credential attack, evasion, or mass scanning capability.
